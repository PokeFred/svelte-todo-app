# How to setup

```bash
$ git clone https://github.com/PokeFred/svelte-todo-app.git
$ cd svelte-todo-app
$ npm install

// development mode
$ npm run dev

// production mode
$ npm run build
$ npm run preview
```

# Frameworks/Libaries

-   Svelte
    -   Version: Typescript
    -   Powered by Vite
-   TailwindCSS
-   FontAwesome
    -   Wrapper: svelte-awesome
-   Prettier
